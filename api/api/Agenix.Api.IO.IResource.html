Declaration

Property Value

A description for this resource.
Remarks

The description is typically used for diagnostics and other such logging when working with the resource.

Implementations are also encouraged to return this value from their ToString() method.

Exists

Does this resource actually exist in physical form?

Declaration
bool Exists { get; }
Property Value
Type Description
bool

true if this resource actually exists in physical form (for example on a filesystem).
Remarks

An example of a resource that physically exists would be a file on a local filesystem. An example of a resource that does not physically exist would be an in-memory stream.

IsOpen

Does this resource represent a handle with an open stream?

Declaration
bool IsOpen { get; }
Property Value
Type Description
bool

true if this resource represents a handle with an open stream.
Remarks

If true, the Stream cannot be read multiple times, and must be read and then closed to avoid resource leaks.

Will be false for all usual resource descriptors.
