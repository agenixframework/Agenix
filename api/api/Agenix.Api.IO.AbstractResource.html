CreateRelative(string)

Creates a new resource that is relative to this resource based on the supplied resourceName.

Declaration
public virtual IResource CreateRelative(string resourceName)
Parameters
Type Name Description
string resourceName

The name of the resource to create.
Returns
Type Description
IResource

The relative resource.
Remarks

This method can accept either a fully qualified resource name or a relative resource name as it's parameter.

A fully qualified resource is one that has a protocol prefix and all elements of the resource name. All other resources are treated as relative to this resource, and the following rules are used to locate a relative resource:

  • If the resourceName starts with '..', the current resource path is navigated backwards before the resourceName is concatenated to the current ResourcePath of this resource.
  • If the resourceName starts with '/', the current resource path is ignored and a new resource name is appended to the RootLocation of this resource.
  • If the resourceName starts with '.' or a letter, a new path is appended to the current ResourcePath of this resource.
Exceptions
Type Condition
UriFormatException

If the process of resolving the relative resource yielded an invalid URI.
NotSupportedException

If this resource does not support the resolution of relative resources (as determined by the value of the SupportsRelativeResources property).
