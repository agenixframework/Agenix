Class TypeRegistry

Provides access to a central registry of aliased Types.

public sealed class TypeRegistry
Remarks

Simplifies configuration by allowing aliases to be used instead of fully qualified type names.

Comes 'pre-loaded' with a number of convenience alias' for the more common types; an example would be the 'int' (or 'Integer' for Visual Basic.NET developers) alias for the int type.

RegisterType(string, string)

Registers an alias for the specified Type.

Declaration
public static void RegisterType(string alias, string typeName)
Parameters
Type Name Description
string alias

A string that will be used as an alias for the specified Type.
string typeName

The (possibly partially assembly qualified) name of the Type to register the alias for.
Remarks

This overload does eager resolution of the Type referred to by the typeName parameter. It will throw a TypeLoadException if the Type referred to by the typeName parameter cannot be resolved.

Exceptions
Type Condition
ArgumentNullException

If either of the supplied parameters is null or contains only whitespace character(s).
TypeLoadException

If the Type referred to by the supplied typeName cannot be loaded.
