Class ExecuteSqlQueryAction

Action executes SQL queries and offers result set validation. The class enables you to query data result sets from a database. Validation will happen on column basis inside the result set.

ValidateSingleValue(string, string, string?, TestContext)

Validates a single value from a database query result set against a specified control value. This method checks if the control value is ignored, a matcher expression, or if it matches the expected value.
