ReceiveMessageAction

This action receives messages from a service destination. Action uses a Endpoint to receive the message; this means that this action is independent of any message transport. The received message is validated using a MessageValidator supporting expected control message payload and header templates.

ReceiveMessageAction.Builder

ReceiveMessageAction.ReceiveMessageActionBuilderSupport

Provides builder support functionalities for constructing ReceiveMessageAction instances.

ReceiveMessageAction.ReceiveMessageActionBuilder<T, TM, TB>

Builder for constructing ReceiveMessageAction instances. This builder provides methods to set various configurations such as validators, selectors, and timeouts for receiving messages. It is designed to support a fluent interface, enabling method chaining to configure and build the action.

ReceiveTimeoutAction

Action expecting a timeout on a message destination, this means that no message should arrive at the destination.

ReceiveTimeoutAction.Builder

SendMessageAction

This action sends a message to a specified message endpoint. The action holds a reference to an Endpoint, which is capable of the message transport implementation. So the action is independent of the message transport configuration.

SendMessageAction.Builder

Builder for constructing and sending message actions.

SendMessageAction.SendMessageActionBuilderSupport

Builder support specifically designed for constructing and sending message actions within the context of a SendMessageAction.

SendMessageAction.SendMessageActionBuilder<T, TM, TB>

Base send message action builder also used by subclasses of base send message action.

SleepAction

StopTimeAction
