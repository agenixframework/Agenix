Class Instrumented.InstrumentedBuilder<T>

Represents a builder for creating and configuring instrumented instances of a specified type. Allows customization of constructor parameters to dynamically instantiate objects. This class is typically used to facilitate dependency injection or dynamic testing scenarios.
