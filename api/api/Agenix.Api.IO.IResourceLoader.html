Interface IResourceLoader

Describes an object that can load IResources.

Namespace: Agenix.Api.IO
Assembly: Agenix.Api.dll
Syntax
public interface IResourceLoader
Remarks

The ConfigurableResourceLoader class is a standalone implementation that is usable outside an Agenix.Context.IApplicationContext; the aforementioned class is also used by the Agenix.Core.IO.ResourceConverter class.
