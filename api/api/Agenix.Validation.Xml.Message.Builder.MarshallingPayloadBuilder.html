public class MarshallingPayloadBuilder : DefaultPayloadBuilder, IMessagePayloadBuilder
Remarks

This class supports the use of either custom implementations of marshallers or named marshallers for transforming model data into the required format. It extends DefaultPayloadBuilder to provide additional functionality specific to marshalling operations.
