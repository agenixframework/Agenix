Remarks

This class integrates with custom resource references and loaded JSON schemas, providing a framework to use them within validation functionalities. It is designed to be initialized with valid resources and schemas for proper operation.

Remarks

This method retrieves the JSON schema from an IResource instance using its GetReader() method and processes it using the JSchema.Load method provided by the Newtonsoft.Json.Schema library.
