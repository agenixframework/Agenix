SetGlobalVariables(GlobalVariables)

Copies the passed globalVariables and adds them to the test context. If any of the copied global variables contain dynamic content (references to other global variables or functions) then this is resolved now. As a result it is important setFunctionRegistry(FunctionRegistry) is called first before calling this method.
