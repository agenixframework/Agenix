Syntax
public class ConfigSectionResource : AbstractResource, IResource, IInputStreamSource
Remarks

If created with the name of a configuration section, then all methods aside from the description return null, false, or throw an exception. If created with an XmlElement, then the InputStream property will return a corresponding Stream to parse.
