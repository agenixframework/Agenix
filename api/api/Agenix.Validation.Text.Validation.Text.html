Namespace Agenix.Validation.Text.Validation.Text

Classes

BinaryBase64MessageValidator

Message validator automatically converts received binary data message payload to base64 String. Assumes control message payload is also base64 encoded String, so we can compare the text data with normal plain text validation.

GzipBinaryBase64MessageValidator

Validates messages that are encoded in Base64, compressed with GZIP, and represented as a byte array payload.

PlainTextMessageValidator

The PlainTextMessageValidator class is responsible for validating plain text messages. It extends the DefaultMessageValidator and provides specific logic to determine if a message of type plaintext is supported.
