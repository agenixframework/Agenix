Syntax
public class UrlResource : AbstractResource, IResource, IInputStreamSource
Remarks

Obviously supports resolution as a Uri, and also as a FileInfo in the case of the "file:" protocol.

Examples

Some examples of the strings that can be used to initialize a new instance of the UrlResource class include...

  • file:///Config/objects.xml
  • http://www.mycompany.com/services.txt
