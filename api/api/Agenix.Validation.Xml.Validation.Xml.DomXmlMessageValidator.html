Class DomXmlMessageValidator

The DomXmlMessageValidator class provides functionality for validating XML messages in a DOM-based manner. It extends the abstract class AbstractMessageValidator<T> specifically for the XmlMessageValidationContext validation context.

Remarks

DomXmlMessageValidator uses XML schema validation to ensure that messages conform to the expected structure and definitions. The validator supports namespace validation and evaluates both message content and schema correctness.
