Syntax
public abstract class AbstractDatabaseConnectingTestAction : AdoDaoSupport, IInitializingObject, ITestAction, INamed, IDescribed
Remarks

This class provides the necessary setup for connecting to a database and executing SQL statements within a specified transaction context. It requires a subclass to implement the execution logic through the DoExecute method.
