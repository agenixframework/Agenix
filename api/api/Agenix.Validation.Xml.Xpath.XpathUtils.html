ReplaceDynamicNamespaces(string, Dictionary<string, string>)

Replaces all dynamic namespaces in an XPath expression with respective prefixes in a namespace map. XPath: 

/{http://sample.org/foo}foo/{http://sample.org/bar}bar
results in 
/ns1:foo/ns2:bar
where the namespace map contains ns1 and ns2.
