Parameters
Type Name Description
string target

The string to check, may be null.
Returns
Type Description
bool

true if the target is not null, Length > zero (0), and does not consist solely of whitespace.
Remarks

More specifically, returns true if the string is not null, it's Length is > zero (0), and it has at least one non-whitespace character.

Examples
StringUtils.HasText(null) = false
StringUtils.HasText("") = false
StringUtils.HasText(" ") = false
StringUtils.HasText("12345") = true
StringUtils.HasText(" 12345 ") = true
