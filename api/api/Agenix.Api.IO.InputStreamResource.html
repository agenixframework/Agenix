Remarks

Should only be used if no other IResource implementation is applicable.

In contrast to other IResource implementations, this is an adapter for an already opened resource - the IsOpen therefore always returns true. Do not use this class if you need to keep the resource descriptor somewhere, or if you need to read a stream multiple times.
