RegisterLoadedAssemblyResolver()

Registers assembly resolver that iterates over the assemblies loaded into the current AppDomain in order to find an assembly that cannot be resolved.

Declaration
public static void RegisterLoadedAssemblyResolver()
Remarks

This method has to be called if you need to serialize dynamically generated types in transient assemblies, such as Spring AOP proxies, because standard .NET serialization engine always tries to load assembly from the disk.
