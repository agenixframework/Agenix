Interface ITask

Represents a task to be performed by an actor within the screenplay design pattern.

Inherited Members
IPerformable.PerformAs<T>(T)

Namespace: Agenix.Screenplay
Assembly: Agenix.Screenplay.dll
Syntax
public interface ITask : IPerformable
Remarks

The ITask interface defines a structure for creating reusable and composable tasks that actors can perform. It extends the IPerformable interface and provides factory methods for creating anonymous tasks, task builders, and other performable operations.
