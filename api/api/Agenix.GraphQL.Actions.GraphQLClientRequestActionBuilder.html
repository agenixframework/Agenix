Class GraphQLClientRequestActionBuilder

Provides functionality for building GraphQL client requests with customizable attributes such as GraphQL query, mutation, subscription, variables, and more. This builder extends the capabilities of SendMessageAction.Builder by adding GraphQL-specific configurations and behaviors.
