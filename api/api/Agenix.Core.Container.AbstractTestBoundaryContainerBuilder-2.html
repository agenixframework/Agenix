Class AbstractTestBoundaryContainerBuilder<T, S>

AbstractTestBoundaryContainerBuilder is an abstract class that facilitates building test boundary action containers with specific conditions and properties. This builder pattern allows users to specify conditions on test names, test groups, namespaces, system properties, and environment variables, ensuring that the before-test logic runs only when these conditions match.
