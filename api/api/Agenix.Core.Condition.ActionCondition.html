Class ActionCondition

Represents a condition that executes a specified test action and determines if the condition is satisfied based on the result of the action execution.

Remarks

The ActionCondition class extends AbstractCondition and operates on a provided ITestAction. This action is tested to determine if the condition is met. The class provides methods to set and retrieve the action, evaluate satisfaction, and manage success or error messages. It also allows retrieval and management of any exceptions caught during execution.
