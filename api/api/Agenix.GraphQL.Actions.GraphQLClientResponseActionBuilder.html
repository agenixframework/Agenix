Class GraphQLClientResponseActionBuilder

Facilitates the creation and configuration of GraphQL client response actions. This class extends the capabilities of ReceiveMessageAction.Builder to provide specialized support for constructing GraphQL response messages. It allows users to define message properties such as GraphQL data, errors, extensions, and metadata, which are tailored for handling GraphQL interactions.
