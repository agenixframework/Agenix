Enum XPathExpressionResult

Enumeration representing the possible result types for XPath expression evaluation. In Agenix XPath expressions a prefix may determine the result type like this: string://MyExpressionString/Value number://MyExpressionString/Value boolean://MyExpressionString/Value The result type prefix is supposed to be stripped off before expression evaluation and determines the evaluation result.
